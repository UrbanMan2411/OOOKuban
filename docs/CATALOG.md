# Каталог артикулов ООО «КубаньБытХим»

Сводный реестр всех артикулов (SKU), найденных в репозитории, с привязкой к источникам.
Машиночитаемая версия — [`docs/catalog.json`](./catalog.json).

> **Важно:** это статические данные из кода репозитория (лендинги, дек, презентации).
> Живой полный каталог маркетплейсов (WB + Ozon) в репо не хранится — он собирается
> скриптом `cockpit/gen_catalogs.mjs` по API с токенами с прод-сервера
> (результат: `https://app.greenpanda-eco.ru` → `/catalogs/wb`, `/catalogs/ozon` + `opisaniya.xlsx`).

## 4. Где какие артикулы живут (карта источников)

| Источник | Что содержит |
|---|---|
| `gp-variants/fullbleed/index.html` → `PRODUCTS` | 6 боевых SKU GreenPanda (905xx) с ШК и описаниями |
| `gp-variants/matreshka-landing/index.html` → `PRODUCTS` | 6 зонтичных SKU Matrёshka (601xx) с EAN |
