# Telegram-магазин КубаньБытХим

Витрина-Mini App + админка внутри cockpit. Один проект Vercel, общие `/api` и Blob.

## Что где
- **Витрина** (покупатели): `/shop` → `shop.html`. Отдельный бандл, без пароля,
  вход через Telegram. Экраны: каталог → карточка → корзина → оформление → оплата.
- **Админка** (вы): дашборд → раздел **«Магазин»** (`/store`), под паролем. Вкладки:
  Каталог · Заказы · Настройки.
- **Каталог (гибрид):** каркас + остатки из WB/Ozon API, цены/штрих-коды/короб из прайса
  (`Импорт прайса`), фото из папки **Карточки** (`cards/`) по артикулу. Дедуп по артикулу.
  Кэш в Blob `_shop/catalog.json`, пересборка кнопкой **Синхронизировать**.
- **Режимы цен:** розница и опт (тумблер в витрине). Опт = ручная опт-цена в каталоге,
  иначе розница минус «Опт-скидка %» из настроек.

## API
| Endpoint | Доступ | Назначение |
|---|---|---|
| `GET /api/shop/catalog` | публичный | витрина: товары + публичные настройки |
| `GET /api/shop/settings` | публичный | публичные настройки магазина |
| `POST /api/shop/order` | публичный (Telegram initData) | оформить заказ / создать инвойс |
| `POST /api/shop/bot` | вебхук Telegram | /start, /id, оплата |
| `POST /api/shop/admin/sync` | под паролем | пересобрать каталог |
| `GET·POST /api/shop/admin/catalog` | под паролем | каталог + правки (скрыть/цена/фото) |
| `GET·POST /api/shop/admin/pricelist` | под паролем | импортированный прайс |
| `GET·POST /api/shop/admin/orders` | под паролем | заказы + статусы |
| `GET·POST /api/shop/admin/settings` | под паролем | настройки магазина |

## Запуск в прод (по шагам)

1. **Env в Vercel** (Project `cockpit` → Settings → Environment Variables). Минимум для
   работы каталога/заказов уже есть (`BLOB_READ_WRITE_TOKEN`, `WB_TOKEN`, `OZON_*`).
   Для оплаты добавьте `TG_BOT_TOKEN`, `TG_PROVIDER_TOKEN` (см. `.env.example`).
2. **Деплой:** `npx vercel --prod` (из папки `cockpit`, проект уже залинкован).
3. **Бот:** @BotFather → `/newbot` → получите токен → в Vercel `TG_BOT_TOKEN` → редеплой.
4. **Привязать витрину к боту:** @BotFather → `/mybots` → Bot Settings → Menu Button →
   URL `https://<домен>/shop`.
5. **Вебхук бота:**
   `https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://<домен>/api/shop/bot`
   (если задали `TG_WEBHOOK_SECRET` — добавьте `&secret_token=<секрет>`).
6. **chat_id для заказов:** напишите боту `/id`, вставьте число в Настройки → «chat_id для заказов».
7. **Оплата (ЮKassa):** @BotFather → `/mybots` → Payments → подключите ЮKassa → токен в
   `TG_PROVIDER_TOKEN` → редеплой. После этого кнопка в витрине станет «Оплатить».

## Наполнение каталога
1. Админка → Магазин → Каталог → **Импорт прайса (xlsx)** (тот же формат, что в генераторе прайсов).
2. Загрузите фото в папку **Карточки** (раздел «Файлы»), имена файлов = артикул.
3. **Синхронизировать WB + Ozon** — соберёт каталог с остатками и фото.
4. При необходимости: скрыть/показать товар, переопределить розн./опт-цену, сохранить.

## Демо-режим
Без `TG_BOT_TOKEN`/`TG_PROVIDER_TOKEN` витрина полностью работает (каталог, корзина,
оформление), заказ сохраняется и (при заданном боте+chat_id) падает вам в Telegram —
но без онлайн-оплаты. Включение оплаты = только добавить токены и редеплой, код менять не нужно.
